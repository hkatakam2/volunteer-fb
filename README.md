# volunteer-fb
    
    To get started, fork this repository to your local machine.
    
    ## Setup Instructions
    
    1. Clone your forked repository to your local machine and navigate into the directory.
    2. Install the necessary dependencies by running:
       `npm install`
    3. Start the development server by running:
       `npm run dev`
    
    ## Contributing
    Pull requests are welcome. For major changes, please open an issue first to discuss what you would like to change.
